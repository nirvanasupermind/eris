# Eris

**This interpreter is in an early stage.**

C++ interpreter for the Eris programming language with a header-only implementation. 

# Synopsis
```
class Point {
    def constructor(self, x, y) {
        self.x = x;
        self.y = y;
    }

    def calc(self) {
        return self.x + self.y;
    }
}

let p = Point(10, 20);
print p.calc(); // 30
```
